# SpaceX Launch Dashboard

Ce projet est une application interactive développée avec **Plotly Dash**. 
Il permet de visualiser et d'analyser les données de lancement de SpaceX.

## Fonctionnalités :
- **Dropdown :** Sélectionner un site de lancement spécifique ou voir tous les sites.
- **Pie Chart :** Analyse du taux de succès par site.
- **Range Slider :** Filtrage des données par masse de charge utile (Payload).
- **Scatter Plot :** Corrélation entre la masse de la charge utile et le succès du lancement par type de booster.

## Technologies utilisées :
- Python
- Plotly & Dash
- Pandas
